Testbed: Jest Basics

  • Installation 1: npm i -D @types/jest jest jest-environment-jsdom
  • Installation 2: "karma" durch "jest" in angular.json ersetzen: `@angular-devkit/build-angular:jest`
  • Installation 3: Karma und Jasmine deinstallieren: npm uninstall @types/jasmine jasmine-core karma karma-chrome-launcher karma-coverage karma-jasmine karma-jasmine-html-reporter

Im Anschluss funktioniert Jest größtenteils wie Karma, nur das Jest deutlich schneller ist, da es Tests parallel ausführen kann und kein echten Webbrowser benötigt.

Auf der anderen Seite unterstützt Jest nicht alle nativen Browser Funktionen/APIs, wie ResizeObserver, LocalStorage usw.
